## Sprint Template Structure

Each sprint includes:
- **Focus Area** — The core skill being developed
- **Duration** — 6 or 8 weeks
- **Linked Rep Types** — Which Real Reps count toward this sprint
- **Weekly Target** — Expected rep frequency
- **Behavior Goal** — Concrete, observable behavior change
- **Observer Question** — What observers are asked monthly
- **Unlocks** — Rewards earned on completion

## Sprint 1: Direct Feedback Mastery

**Focus Area:** `direct-feedback`  
**Duration:** 8 weeks  
**Difficulty:** Intermediate  
**Prerequisites:** Foundation complete

### The Challenge

> Stop avoiding hard conversations. Deliver redirecting feedback within 24 hours of any issue — before it festers.

### Description

Most leaders wait too long to give corrective feedback. By the time they speak up, the issue has compounded and emotions run higher. This sprint builds the muscle of **immediate, direct feedback** — addressing issues while they're small and manageable.

### Behavior Goal

"I will deliver redirecting feedback within 24 hours of observing an issue, instead of waiting or hoping it resolves itself."

### Linked Rep Types

| Rep Type | Description |
|----------|-------------|
| **DRE** | Deliver Redirecting Feedback |
| **DSC** | Difficult Conversations |

### Weekly Target

2-3 Real Reps per week

### Observer Pulse Question

**Primary Question:**  
"In the past month, how often did you observe [Leader] giving clear, direct feedback when something needed to be addressed?"

**Scale:**
- Never observed
- Rarely (1-2 times)
- Sometimes (3-4 times)
- Often (5+ times)
- Consistently (this is now typical behavior)

**Follow-up (optional):**  
"Any specific example of [Leader] giving effective (or ineffective) direct feedback?"

### Unlocks on Completion

| Type | Unlock | Description |
|------|--------|-------------|
| Advanced Rep | De-escalating High Emotion | Handle feedback when the recipient becomes emotional |
| Advanced Rep | Letting Someone Go | The hardest conversation |
| Situation Room | Termination Prep | Crisis prep for termination conversations |
| Badge | Direct Feedback Master | Profile badge |

### AI Coaching Notes

When recommending this sprint, highlight:
- Pattern of DRE avoidance in Foundation
- Long gaps between issue observation and feedback delivery
- Self-assessment comments about "waiting to see if it improves"

## Sprint 2: Difficult Conversations

**Focus Area:** `difficult-conversations`  
**Duration:** 8 weeks  
**Difficulty:** Advanced  
**Prerequisites:** Foundation complete, Direct Feedback Mastery recommended

### The Challenge

> Stop avoiding the conversations that keep you up at night. Initiate one difficult conversation per week that you've been putting off.

### Description

Every leader has a list of conversations they're avoiding — the underperformer, the conflict between team members, the pushback to their boss. This sprint is about **clearing the backlog** of difficult conversations and building confidence in navigating high-stakes dialogue.

### Behavior Goal

"I will initiate one difficult conversation per week that I've been putting off, rather than waiting for the 'right time' that never comes."

### Linked Rep Types

| Rep Type | Description |
|----------|-------------|
| **DSC** | Difficult Conversations |
| **DRE** | Deliver Redirecting Feedback |
| **MNG-UP** | Managing Up (if unlocked) |

### Weekly Target

1-2 Real Reps per week (these are harder, so lower volume)

### Observer Pulse Question

**Primary Question:**  
"In the past month, have you seen [Leader] initiate difficult conversations that others might avoid?"

**Scale:**
- No, they seem to avoid hard topics
- Occasionally, but they still hesitate
- Yes, they're willing to have tough talks
- Yes, they proactively address difficult issues
- Absolutely — they lean into hard conversations

**Follow-up (optional):**  
"Is there a difficult conversation you think [Leader] is still avoiding?"

### Unlocks on Completion

| Type | Unlock | Description |
|------|--------|-------------|
| Advanced Rep | Navigating Peer Conflict | Mediate conflict without authority |
| Advanced Rep | Delivering Bad News | Layoffs, budget cuts, project cancellations |
| Situation Room | High-Stakes Meeting Prep | Board meetings, executive presentations |
| Badge | Difficult Conversation Navigator | Profile badge |

### AI Coaching Notes

When recommending this sprint, highlight:
- Few DSC reps attempted in Foundation
- Debrief comments about "waiting for things to improve"
- Avoidance patterns in skip-level or peer situations

## Sprint 3: Delegation & Empowerment

**Focus Area:** `delegation`  
**Duration:** 6 weeks  
**Difficulty:** Intermediate  
**Prerequisites:** Foundation complete

### The Challenge

> Stop being the bottleneck. Delegate meaningful work — not just tasks — and resist the urge to take it back.

### Description

Many leaders delegate tasks but retain decision-making authority, creating bottlenecks and disempowered teams. This sprint focuses on **true delegation** — giving others ownership of outcomes, not just activities, and coaching them through challenges instead of solving problems for them.

### Behavior Goal

"I will delegate at least one significant decision or project per week to a team member, provide context instead of instructions, and resist the urge to intervene unless asked."

### Linked Rep Types

| Rep Type | Description |
|----------|-------------|
| **SCE** | Set Clear Expectations |
| **DEL** | Delegation (new type for Ascent) |
| **DRF** | Deliver Reinforcing Feedback (for coaching moments) |

### Weekly Target

2 Real Reps per week

### Observer Pulse Question

**Primary Question:**  
"In the past month, how often did [Leader] delegate meaningful work and let you own the outcome?"

**Scale:**
- Not at all — they hold onto everything
- Rarely — they delegate tasks but not decisions
- Sometimes — they're trying to let go more
- Often — they give me real ownership
- Consistently — I feel empowered to make decisions

**Follow-up (optional):**  
"What's one thing [Leader] could delegate more of?"

### Unlocks on Completion

| Type | Unlock | Description |
|------|--------|-------------|
| Advanced Rep | Delegating to Senior Staff | When the delegate is more experienced |
| Advanced Rep | Letting Go of Your Baby | Delegating work you created |
| Badge | Empowerment Leader | Profile badge |

### AI Coaching Notes

When recommending this sprint, highlight:
- Leader profile indicates large team or new to management
- Patterns of micromanagement in debrief comments
- Feedback about being a bottleneck

## Sprint 4: Strategic Communication

**Focus Area:** `strategic-communication`  
**Duration:** 6 weeks  
**Difficulty:** Intermediate  
**Prerequisites:** Foundation complete

### The Challenge

> Stop assuming your team knows what you know. Share the "why" behind decisions and connect daily work to bigger goals.

### Description

Leaders often make decisions with context their teams don't have, then wonder why people don't execute with the same urgency or alignment. This sprint builds the habit of **sharing strategic context** — explaining the "why," connecting work to outcomes, and helping teams see the bigger picture.

### Behavior Goal

"I will share strategic context in every team interaction — meetings, 1:1s, Slack messages — so my team understands not just what to do, but why it matters."

### Linked Rep Types

| Rep Type | Description |
|----------|-------------|
| **SCE** | Set Clear Expectations |
| **STR** | Strategic Communication (new type for Ascent) |
| **DRF** | Deliver Reinforcing Feedback (when team shows strategic thinking) |

### Weekly Target

3 Real Reps per week (these can be quick moments)

### Observer Pulse Question

**Primary Question:**  
"In the past month, how well did [Leader] help you understand the 'why' behind decisions and priorities?"

**Scale:**
- Not at all — I often don't know why we're doing things
- Rarely — context is shared inconsistently
- Sometimes — I get some background
- Often — I understand the reasoning behind most decisions
- Consistently — I always understand how my work connects to bigger goals

**Follow-up (optional):**  
"What's one decision or priority you wish [Leader] had explained better?"

### Unlocks on Completion

| Type | Unlock | Description |
|------|--------|-------------|
| Advanced Rep | Pitching Strategy | Persuading stakeholders on strategic direction |
| Advanced Rep | Communicating Change | Helping teams navigate organizational shifts |
| Situation Room | All-Hands Prep | Preparing for company-wide announcements |
| Badge | Strategic Communicator | Profile badge |

### AI Coaching Notes

When recommending this sprint, highlight:
- SCE reps focused on "what" without "why"
- Debrief comments about team confusion or misalignment
- Leader managing a distributed or growing team

## Sprint 5: Building Trust

**Focus Area:** `building-trust`  
**Duration:** 8 weeks  
**Difficulty:** Foundational  
**Prerequisites:** Foundation complete

### The Challenge

> Trust isn't built in big moments — it's built in small ones. Show up consistently: follow through, be vulnerable, give credit.

### Description

Trust is the foundation of every leadership relationship. This sprint focuses on the **micro-behaviors** that build (or erode) trust over time: following through on commitments, admitting mistakes, giving credit to others, and being transparent about challenges.

### Behavior Goal

"I will actively build trust through consistent micro-behaviors: always follow through, admit when I'm wrong, give credit publicly, and share challenges honestly with my team."

### Linked Rep Types

| Rep Type | Description |
|----------|-------------|
| **DRF** | Deliver Reinforcing Feedback |
| **TRS** | Trust-Building Moments (new type for Ascent) |
| **DSC** | Difficult Conversations (vulnerability builds trust) |

### Weekly Target

2-3 Real Reps per week

### Observer Pulse Question

**Primary Question:**  
"In the past month, how would you rate your trust in [Leader]?"

**Scale:**
- Low — I'm cautious about what I share with them
- Developing — Trust is growing but not solid yet
- Moderate — I trust them in most situations
- High — I trust them with important matters
- Very High — I fully trust their intentions and follow-through

**Follow-up (optional):**  
"What's one thing [Leader] could do to build more trust with you?"

### Unlocks on Completion

| Type | Unlock | Description |
|------|--------|-------------|
| Advanced Rep | Rebuilding Broken Trust | Repair relationships after a breach |
| Advanced Rep | Inheriting a Team | Building trust with a new team from day one |
| Badge | Trust Builder | Profile badge |

### AI Coaching Notes

When recommending this sprint, highlight:
- New leader or recently promoted
- Feedback about being "hard to read" or "unpredictable"
- Team survey data showing trust gaps
- Leader managing remote or distributed team

### Sprint Template Fields

```javascript
// For Content Manager implementation
{
  id: string,                    // Unique identifier
  name: string,                  // Display name
  focusArea: string,             // Focus area key
  duration: number,              // Weeks (6 or 8)
  difficulty: string,            // beginner | intermediate | advanced

  linkedRepTypes: string[],      // Rep types that count
  weeklyRepTarget: number,       // Expected reps per week
  
  pulseQuestion: string,         // Primary observer question
  pulseScale: string[],          // 5-point scale options
  pulseFollowUp: string,         // Optional follow-up question
  
  unlocks: [                     // Completion rewards
    { type: string, id: string, description: string }
  ],
  
  prerequisites: {
    foundationComplete: boolean,
    priorSprints: string[]       // Sprint IDs required first
  },
  
  aiCoachingNotes: string,       // Guidance for recommendations
  
  status: string,                // draft | active | archived
  sortOrder: number
}
```
