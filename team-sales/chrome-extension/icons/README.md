# Extension Icons

This folder should contain the extension icons in PNG format:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extensions management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Design Specifications

The icons should use the LeaderReps brand colors:
- **Primary Background**: `#47A88D` (Corporate Teal)
- **Text/Logo**: White `#FFFFFF`
- **Border/Accent**: `#002E47` (Corporate Navy)

## Suggested Design

A simple "LR" monogram or a stylized people/network icon on a teal rounded square background.

## Generating Icons

You can use tools like:
- Figma
- Adobe Illustrator
- Canva
- Online icon generators

Or convert the inline SVG from popup.html to PNG at different sizes.
